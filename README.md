# Library Management System

## 📌 Project Description

This is a Spring Boot web application developed to manage a library system. It supports basic CRUD operations for managing authors and books.

## ⚙️ Technologies Used

* Spring Boot
* Spring Data JPA (Hibernate)
* JSP & JSTL
* MySQL Database
* Maven

## 🚀 Features

* Add new authors and books
* View list of authors and books
* Update existing records
* One-to-Many relationship between Author and Book
* Custom JPQL query using JOIN FETCH

## ▶️ How to Run

1. Clone the repository
2. Open the project in Eclipse or IntelliJ
3. Configure MySQL database
4. Run `LibraryApplication.java`
5. Open browser and go to:
   http://localhost:8080
